# Inkhaven 1.2.9 — in flight

The 1.2.9 cycle picks up the items 1.2.8 left on the
table and continues the polish pass.  1.2.8 was the
"new surfaces" release (embedded shell + HJSON editor +
markdown Help book); 1.2.9 will likely lean closer to
polish + bugfixes again before 1.3 lands the split-view
rework.

This document is a living scratchpad — entries land here
as the work lands.  Each item starts as a one-line plan;
gets fleshed out when the corresponding feature ships;
gets a chord / config / hook callout when relevant.

## Planned themes

### Split view — the deferred 1.3 work

1.3 has been "next release" since 1.2.5.  The reason it
keeps slipping is that each polish cycle finds enough
small items to fill the branch.  1.2.9 should be the
last polish cycle before this happens — once 1.3 lands,
the F4 split-edit workflow + a real two-pane editor +
side-by-side snapshot diff land together.

### Carryovers from 1.2.8

The 1.2.8 release notes listed three items that didn't
ship:

- **Cascade-impact AI on event re-time.** Optional
  `Ctrl+V Shift+I` extension that flags the linked-
  paragraph fallout when an event is moved.
- **F11-on-paragraph "suggest event" variant.** Propose
  a draft event from the prose under the cursor.
- **Visual editor overlay for `Ctrl+V h`.** Tab +
  trailing-whitespace glyphs without breaking the
  column-bookkeeping that match + lexicon highlights
  depend on.
