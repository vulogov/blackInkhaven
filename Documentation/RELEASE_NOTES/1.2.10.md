# Inkhaven 1.2.10 — in flight

The 1.2.10 cycle picks up the items 1.2.9 left on
the table and continues the polish + prose-craft
arc.  1.2.9 was the "audit your prose while you
write" release (filter words + repeated phrases +
show-don't-tell + concordance + sentence rhythm +
POV chip + streak heatmap + scene-break nav + TTS).
1.2.10 will likely lean closer to polish + bugfixes
again before 1.3 lands the split-view rework.

This document is a living scratchpad — entries land
here as the work lands.  Each item starts as a one-
line plan; gets fleshed out when the corresponding
feature ships; gets a chord / config / hook callout
when relevant.

## Planned themes

### Carryovers from 1.2.9

- **AI-driven sentence-rhythm rewrite** — a chord
  that sends the open paragraph to the LLM with a
  prompt asking it to break a MONOTONE verdict.
  Pairs with `Ctrl+B Shift+H` the same way the AI
  show-don't-tell scan pairs with the regex
  overlay.
- **Per-language emotion + linking-verb lists**
  for Russian / French / German / Spanish in the
  show-don't-tell detector.  English ships
  built-in; the others have empty scaffolds — the
  community + dogfooding fills them in this cycle.
- **Concordance: jump-to-occurrence on Enter**.
  Currently the modal is read-only; pressing
  Enter on a row should close the modal and
  open the source paragraph at the line of the
  first sample.

### Split view — the deferred 1.3 work

1.3 has been "next release" since 1.2.5.  It keeps
slipping because each polish cycle finds enough
single-pane work to fill itself.  If 1.2.10 is
genuinely a quiet cycle, the split-view rework
lands here.

### Other candidates

- Mouse-driven scene-break selection (drag from
  one break-line to the next selects the scene).
- Concordance export — `inkhaven export
  concordance` for the CSV / JSON variant of the
  modal data, so users can pipe it into their own
  analysis.
- Show-don't-tell theme audit — per-detector
  underline style override (DIM / BOLD /
  REVERSED / NONE), since the teal underline
  reads differently on different terminal
  palettes.
