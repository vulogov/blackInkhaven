# Inkhaven 1.2.11 — in flight

1.2.10 was the "standalone editors" release —
`inkhaven config` and `inkhaven prompts-editor` both
shipped a dedicated TUI for their respective HJSON
files.  1.2.11 picks up the carryovers from 1.2.10
and the deferred items from 1.2.9.

This document is a living scratchpad — entries land
here as the work lands.  Each item starts as a one-
line plan; gets fleshed out when the corresponding
feature ships; gets a chord / config / hook callout
when relevant.

## Planned themes

### Carryovers from 1.2.10

  * **Apply prompts-editor critique back into the
    main TUI's flows.**  The reviewer-LLM can
    suggest a rewrite that survives Ctrl+B G into
    the editor — but the wider review-loop ends at
    Ctrl+S to disk.  Consider piping the same
    review-as-you-edit shape into the main TUI's
    `Ctrl+B 0` HJSON editor for inkhaven.hjson.

  * **HSL slider modal for the colour widget.**
    Hex + RGB-swatch ships today; an HSL slider
    overlay would make picking accessible colours
    easier without leaving the TUI.

  * **Actual file-picker for the path widget.**
    Text input + existence check ships today;
    integrate the main TUI's F3 file picker.

  * **More enum entries.**  The metadata table
    currently routes `typst_compile.engine` and
    `embeddings.model`.  Audit every `String`
    field in `Config` for fixed-set candidates
    (themes, output formats, stemmer languages).

  * **Build-time doc-comment extraction polish.**
    The current extractor handles `Option<T>` /
    `Vec<T>` / `HashMap<K, V>` descents; nested
    `Option<HashMap<K, V>>` shapes aren't yet
    handled.  Audit + fix.

  * **AI-driven sentence-rhythm rewrite** (deferred
    from 1.2.9).  A chord that sends the open
    paragraph to the LLM with a prompt asking it
    to break a MONOTONE verdict.  Pairs with
    Ctrl+B Shift+H the same way the AI
    show-don't-tell scan pairs with the regex
    overlay.

  * **Per-language emotion + linking-verb lists**
    for the show-don't-tell detector (Russian /
    French / German / Spanish — English ships
    built-in; the others have empty scaffolds).

  * **Concordance: jump-to-occurrence on Enter**.
    Currently the modal is read-only; pressing
    Enter on a row should close the modal and
    open the source paragraph at the line of the
    first sample.

### Split view — the deferred 1.3 work

1.3 has been "next release" since 1.2.5.  It keeps
slipping because each polish cycle finds enough
single-pane work to fill itself.  If 1.2.11 is
genuinely a quiet cycle, the split-view rework
lands here.

### Other candidates

  * Mouse-driven scene-break selection (drag from
    one break-line to the next selects the scene).
  * Concordance export — `inkhaven export
    concordance` for the CSV / JSON variant of
    the modal data.
  * Show-don't-tell theme audit — per-detector
    underline style override (DIM / BOLD /
    REVERSED / NONE), since the teal underline
    reads differently on different terminal
    palettes.
